Web Tier

Web tier works for clients that are in need of technical help in modernizing their websites. Tier members learn new skills and put existing knowledge to use through hands-on experience.

Web Tier takes requests for clubs on campus, building them new websites or redesigning their old ones. Some examples of work we've done in the past include Premed@Berkeley and ISA.

ISA and Premed@Berkeley were both built using Meteor, but web tier members have also used Angular and React in the past. We work together in a team setting from design to implementation.
